Introducció a Angular

Angular és un framework de desenvolupament per a aplicacions web escrit en TypeScript. Desenvolupat per Google, Angular permet crear aplicacions d'una sola pàgina (SPA) robustes i escalables.

{% include info_box.html contingut="Definició: Angular és un framework complet que proporciona una estructura i eines per desenvolupar aplicacions completes, mentre que una biblioteca com React només proporciona funcions específiques." %}

Característiques Principals

  • TypeScript: Llenguatge principal amb tipatge fort
  • Components: Arquitectura basada en components reutilitzables
  • CLI: Eina de línia de comandaments potent
  • Ecosystem complet: Routing, HTTP client, forms integrats

Abans de començar, necessitem configurar el nostre entorn de desenvolupament.
